
## Description

admin 接口部分 使用nest创建
前端为vue3搭建的后台管理 
前端地址 https://github.com/MasayumeKTYA/admin_vue3
nest+mysql+jwt+typeorm
## Installation

```bash
$ yarn install
```

## 运行

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## 测试

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
